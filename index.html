Tecnologias

HTML

HTML, ou Linguagem de Marcação de Hipertexto, é a espinha dorsal de uma página web. Ela define a estrutura e o conteúdo usando uma variedade de elementos e tags predefinidos, criando a base para a experiência digital

CSS

CSS, ou Folhas de Estilo em Cascata, é o toque de estilo que dá vida às páginas web. Ele permite controlar cores, fontes, layout e muito mais, transformando o HTML em designs visualmente cativantes e coesos.

JavaScript

Linguagem de programação por trás da interatividade na web. Esta linguagem dinâmica e versátil permite criar animações, formulários inteligentes, comunicação com servidores e até mesmo jogos, tornando as páginas web mais dinâmicas e envolventes.

PHP

Linguagem poderosa presente na interatividade web. Esta linguagem versátil e dinâmica possibilita criar animações, formulários inteligentes, comunicação com servidores e até jogos, tornando as páginas mais dinâmicas e atrativas.

Canva

Com o Canva, você desbloqueia seu potencial criativo. Esta plataforma de design gráfico online oferece uma ampla gama de ferramentas e recursos para criar conteúdos visuais incríveis, desde mídias sociais até apresentações e convites, tornando o design acessível para todos.

Principais Soft Skills

Escrita

A habilidade de expressar ideias de forma clara e persuasiva por meio de palavras escritas, comunicando efetivamente mensagens e informações para diversos públicos e propósitos.

Trabalho em Equipe

A capacidade de colaborar harmoniosamente com outros indivíduos em busca de objetivos comuns, contribuindo ativamente, ouvindo atentamente e respeitando as opiniões e habilidades de todos os membros da equipe.

Comunicação

A aptidão para transmitir informações de maneira clara, concisa e compreensível, utilizando uma variedade de canais e métodos para garantir que a mensagem seja recebida e entendida pelo destinatário

Resiliência

A habilidade de se adaptar e se recuperar de adversidades, desafios e fracassos, mantendo uma atitude positiva, perseverança e capacidade de enfrentar obstáculos com determinação e confiança.

Aprendizado

A disposição e capacidade de adquirir novos conhecimentos, habilidades e experiências de forma contínua, buscando oportunidades de desenvolvimento pessoal e profissional e refletindo sobre os aprendizados para melhorar constantemente.

Pensamento Criativo

A capacidade de gerar ideias originais e inovadoras, abordar problemas de maneiras não convencionais, e conectar conceitos aparentemente desconexos para encontrar soluções imaginativas e eficazes.

Projetos

Cafeteria IOS

A página Cafeteria IOS é uma Landing Page de informações sobre café. Utilizando técnicas avançadas de parallax, ela cria uma narrativa visual imersiva que transporta os visitantes para uma jornada sensorial pelo mundo dos grãos.

Vincent's Calculator

Vincent's Calculator é um projeto inspirada nas obras de Van Gogh, esta calculadora em JavaScript não apenas realiza cálculos, mas também celebra a arte e a criatividade do renomado pintor.

CAP´S LETTERS

CAP´S LETTERS é um projeto proposto para debater sobre o futuro da programação, com o tema de IA (Inteligência Artificial) e o mercado de trabalho dos Desenvolvedores.

Game of Old Lady

O Game of Old Lady é um projeto de jogo da velha. Desenvolvido em JavaScript, destaca-se por personalizar o tema da partida, proporcionando uma experiência única aos jogadores.

INNOVA

O Projeto INNOVA é uma colaboração destinada a promover soluções inovadoras para desafios sociais específicos. Realizado dentro do Instituto da Oportunidade Social, para o "Projeto Família", onde valorizamos a família e os homenageamos em uma empresa fictícia.

Quiz Soulslike

Quiz Soulslike é um jogo de perguntas e respostas, feito com JavaScript para seu funcionamento, com o tema do gênero de video game "Soulslike".

Sistema Solar

A Landing Page Sistema Solar oferece uma experiência imersiva sobre astronomia. Combinando com HTML, CSS e técnicas de parallax, ela leva os visitantes em uma viagem Sistema Solar.

Gerador de Tabuada

O Gerador de Tabuada é uma ferramenta educacional. Projetado especialmente para o público infantil, utiliza uma abordagem lúdica e interativa para tornar o aprendizado da matemática divertido e envolvente.

To-Do List

To-do list facilita a organização de suas tarefas. Desenvolvido com JavaScript, oferece funcionalidades dinâmicas e lógica eficiente para gerenciar e acompanhar o progresso das atividades diárias de forma intuitiva.

Mugiwara Jump

Mugiwara Jump é jogo de plataforma simples. Inspirado no popular anime "One Piece", desafia os jogadores a explorarem o mundo pirata enquanto saltam obstáculos.

Calcule seu IMC

Calcular seu IMC não se limita a fornecer uma simples medição de índice de massa corporal. Esta Landing Page desenvolvida em JavaScript oferece uma ferramenta para monitorar e melhorar a saúde pessoal, apresentando informações valiosas sobre bem-estar.

Kiko Clicker

Kiko Clicker oferece mais do que apenas uma forma de passar o tempo. Com sua mecânica simples e viciante, este jogo desenvolvido em parceria com meu colega Cauber Streck, desafia os jogadores a aprimorarem suas habilidades de clique enquanto competem por pontuações mais altas.
